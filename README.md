# 📊 Trading Journal Web App

A simple and interactive trading journal built with Python, Streamlit, and Pandas to track trades and analyze performance.

## 🚀 Features
- Log trades with key details  
- Auto Risk/Reward (RR) calculation  
- Performance dashboard (win rate, profit, equity curve)  
- Multi-account support  
- Asset-wise analysis  
- Emotion tracking  

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app.py  

## ☁️ Deployment
Deploy easily on Streamlit Cloud by connecting your GitHub repo.
